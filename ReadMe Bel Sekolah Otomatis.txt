CARA PENGGUNAAN:

SET JADWAL(HARDWARE):
1. Pilih hari dimana jadwal akan diubah.
2. Untuk menambah jadwal tekan tombol "Tambah Jadwal"
3. Masukan data "JAM", "JADWAL" dan "BEL".
4. kemudian tekan "Simpan Jadwal"
5. Untuk menghapus jadwal, harus mengklik jadwal yang ingin dihapus kemudian tekan "Hapus Jadwal"

SET JADWAL (SOFTWARE):
1. Masukan jam pada kolom "JAM" dan pilih "BEL" yang diinginkan.
2. Jika ingin menyamakan jadwal dengan jadwal Hardware tekan tombol "SAMAKAN DENGAN JADWAL HARDWARE"
3. Ketika selesai tekan tombol "JALANKAN JADWAL", jangan matikan komputer ketika jadwal sedang berjalan. 
4. Jika tidak digunakan tekan "MATIKAN JADWAL".

CEK AUDIO (HARDWARE)
1. Pilih JADWAL yang diinginkan.
2. Tekan tombol "CEK AUDIO"

CEK AUDIO (SOFTWARE)
1. Pilih JADWAL yang diinginkan.
2. Tekan tombol "CEK AUDIO"

MEMASUKAN JADWAL KE ALAT:
1. Tancapkan serial ke alat dan komputer anda.
2. Lakukan pemilihan PORT dan Baudrate (9600).
3. Tekan tombol "HUBUNGKAN"
4. Tekan tombol "JALANKAN" untuk mengirim data ke alat.
